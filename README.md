# Fire Web Revival - AVISHKAAR Hackathon Website

A modern, responsive website for AVISHKAAR Season 4 hackathon built with React and Vite.

## Project Overview

This project is a hackathon website featuring:
- Modern UI with fire-themed animations
- Responsive design
- Multiple sections: Hero, About, Themes, Hackathon Info, Gallery, FAQ, and more
- Registration and Login pages
- Team and Tracks pages

## Technologies Used

- **Vite** - Build tool and dev server
- **React** - UI library
- **JavaScript/JSX** - Programming language
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **React Router** - Routing
- **Radix UI** - UI components
- **Lucide React** - Icons

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```

The development server will start at `http://localhost:8080`

### Build for Production

```sh
# Build the project
npm run build

# Preview production build
npm run preview
```
